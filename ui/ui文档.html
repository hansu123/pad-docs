# ui文档

# 1. HTimepicker

时间选择器

# Attributes

参数 说明 类型 可选值 默认-值
value 时间值，需要使用.sync进行双向绑定 Date - -
disabled 是否禁用 Boolean - false
placeholder 占位文本 String - "选择时间"

# 2. HDatePicker

日期选择器

# Attributes

参数 说明 类型 可选值 默认-值
value 日期值，需要使用.sync进行双向绑定 Date - -
disabled 是否禁用 Boolean - false
placeholder 占位文本 String - "选择日期"

# 3. HPicker

移动端选择框

# Attributes

参数 说明 类型 可选值 默认-值
model 指定选中项目的项的 value 值，需要使用.sync进行双向绑定 String，Number - -
options 选项数据，目前只支持[{name:value}]形式 Array - -
disabled 是否禁用 Boolean - false
placeholder 占位文本 String - "请选择"

# 4.HTab

选项卡

# Attributes

参数 说明 类型 可选值 默认-值
value 使用 v-model，初始化时选中对应的下标 Number - -
tabData 选项卡数据 Array - -

# Methods

参数 说明 参数
change 滑动不同的tab触发的方法 选择tab的下标（index）
